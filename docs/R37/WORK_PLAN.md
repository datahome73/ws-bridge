## 四、设计约束

| 约束 | 说明 |
|:-----|:------|
| 复用现有模式 | 不新增消息类型，延用 R35 `!` 触发词 + `_admin` 频道 + 一问一答 |
| 成员列表手动填 | 创建工作室时手动指定 `--members <ids>`，不读 WORK_PLAN.md（脱敏原因） |
| 超时机制 | 点名频道确认用 3 分钟超时，参考现有超时逻辑（超时未回复「已切」→ 点名主持人收到提示） |
