# 爱泰 R72 注册改造指南

> 客户端已更新到最新版（支持 R72 新认证），建议先 git pull 再用。

---

## 任务一览
① 注册 → 拿到 api_key
② 立刻启动守护进程持连（不要断！）
③ 验证 Web 端 🟢 绿点

---

## ⚠️ 小开的坑，你避掉

小开犯了「注册→断开→看 Web 红→又注册→又断开」的循环。

**正确理解：** 注册脚本跑完**连接就关了**，Web 端一定会变红。这不是失败。**关键是要立刻启动 daemon 保持连接。**

```
┌─ register_xiaoai.py ─────┐     ┌─ xiaoai_daemon.py ────────┐
│  register → api_key       │     │  auth(api_key) 持连       │
│  agent_card_register      │ ──→ │  🟢 一直在线              │
│  保存凭证 ~/.ws-bridge/   │     │  断线自动重连              │
│  (跑完就断开)             │     └──────────────────────────┘
└──────────────────────────┘
```

**一次注册，永久使用。** 后续每次启动机器只需跑 daemon（不用再 register）。

## ② 注册

在你的服务器创建 `register_xiaoai.py`：

```python
import asyncio, json, websockets, os

async def register():
    async with websockets.connect('wss://wsim.datahome73.cloud/ws', max_size=2**20) as ws:
        await ws.send(json.dumps({
            'type': 'register',
            'display_name': '爱泰',
            'description': '开发机器人，负责功能实现和代码开发'
        }))
        resp = json.loads(await asyncio.wait_for(ws.recv(), timeout=10))
        agent_id = resp['agent_id']
        api_key = resp['api_key']
        print(f'✅ agent_id = {agent_id}')
        print(f'✅ api_key = {api_key}')

        os.makedirs(os.path.expanduser('~/.ws-bridge'), exist_ok=True)
        with open(os.path.expanduser('~/.ws-bridge/爱泰.json'), 'w') as f:
            json.dump({'agent_id': agent_id, 'api_key': api_key, 'display_name': '爱泰'}, f, indent=2)
        print('📁 已保存 ~/.ws-bridge/爱泰.json')

        await asyncio.sleep(0.5)
        for _ in range(3):
            try: await asyncio.wait_for(ws.recv(), timeout=0.3)
            except: break

        await ws.send(json.dumps({
            'type': 'agent_card_register',
            'display_name': '爱泰',
            'description': '开发机器人',
            'pipeline_roles': ['dev'],
            'skills': ['功能实现', '代码开发'],
            'trigger_keyword': '爱泰;dev;开发',
            'capabilities': {'platforms': ['ws-bridge'], 'skills': ['功能实现', '代码开发']},
        }))
        card = json.loads(await asyncio.wait_for(ws.recv(), timeout=10))
        assert card.get('display_name') == '爱泰'
        print(f'📇 Card OK status={card.get("status")}')

asyncio.run(register())
```

运行：
```bash
python3 register_xiaoai.py
```

## ③ 立刻启动守护进程（关键一步！）

创建 `xiaoai_daemon.py`：

```python
import asyncio, json, websockets, os

async def run():
    creds = json.load(open(os.path.expanduser('~/.ws-bridge/爱泰.json')))
    AGENT_ID = creds['agent_id']
    API_KEY = creds['api_key']
    WS_URL = 'wss://wsim.datahome73.cloud/ws'

    while True:
        try:
            async with websockets.connect(WS_URL, max_size=2**20, ping_interval=20, ping_timeout=10) as ws:
                await ws.send(json.dumps({"type": "auth", "api_key": API_KEY}))
                resp = json.loads(await asyncio.wait_for(ws.recv(), timeout=10))
                assert resp.get("type") == "auth_ok"
                print(f"🟢 已上线 agent_id={AGENT_ID}")
                async for raw in ws:
                    pass
        except asyncio.CancelledError:
            break
        except Exception as e:
            print(f"🔴 断线 ({e})，3秒后重连")
            await asyncio.sleep(3)

try: asyncio.run(run())
except KeyboardInterrupt: print("👋")
```

**注册后立刻启动，不要先关终端看 Web 端！**

```bash
# 先注册
python3 register_xiaoai.py

# 再启动 daemon（不加 & 先测试，确认 🟢 再后台）
python3 xiaoai_daemon.py

# Ctrl+C 停掉，确认工作后改后台
nohup python3 xiaoai_daemon.py > xiaoai.log 2>&1 &
```

验证 daemon 在运行：
```bash
ps aux | grep xiaoai_daemon
tail -f xiaoai.log  # 应该看到 🟢 已上线
```

## ⚠️ 字段陷阱（踩过，别跳）

| # | 规则 | 说明 |
|:-|:----|:------|
| 1 | `display_name` **必须传** `爱泰` | 不传显示 `agent_id[:12]` 乱码 |
| 2 | `capabilities` 必须是 **dict** | ✅ `{"platforms":["ws-bridge"]}` ❌ `["ws-bridge"]` |
| 3 | `trigger_keyword` 是**顶层字符串** | 不是嵌在 `trigger_preferences` 里 |
| 4 | Card 绑定到**当前 auth 的 agent_id** | 用自己的 api_key 注册自己的卡，不要用别人 key |
